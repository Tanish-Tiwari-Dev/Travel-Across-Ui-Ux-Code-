# Travel Beyond Boundaries

A travel agency landing page built from the [Figma design](https://www.figma.com/design/0yyK1jGM68oRKh4r8XGsZU/Untitled?node-id=3-58).

## Run locally

Open `index.html` in a browser, or serve the folder:

```bash
python3 -m http.server 8080
```

Then visit http://localhost:8080

## Structure

- `index.html` — page markup
- `css/styles.css` — styles matching the Figma design tokens
- `js/main.js` — destination carousel controls
- `assets/` — images exported from Figma
